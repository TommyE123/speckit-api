# Data Model: GitHub Actions CI Build Workflow

**Feature**: 007-github-actions-ci
**Status**: N/A

---

This feature introduces a single YAML configuration file (`.github/workflows/build.yml`).
There are no data entities, domain models, persistence schemas, state machines, or validation
rules to design.

No source code changes are permitted by FR-011, and no data layer exists within the CI
workflow itself.

**Decision**: Data model phase skipped — not applicable for pure CI infrastructure.
